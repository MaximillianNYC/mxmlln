# EXP 029 - SVG Transition Test

A Next.js React app implementing an SVG mask transition animation that shrinks from full bleed to a 100px circle.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- SVG mask animation that transitions from full viewport coverage to a 100px circle
- Smooth CSS transition over 1 second
- Blur effect on the mask circle
- Full bleed background image
- Built with Next.js 15 and Tailwind CSS v4
