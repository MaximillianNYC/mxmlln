# EXP 027 - ChatBar Component

A Next.js React app implementing the ChatBar design from Figma.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design

This component implements the ChatBar design from Figma with:
- Dark theme with custom color variables
- Rounded borders and proper spacing
- Logo/image row at the top
- Body section for chat content
